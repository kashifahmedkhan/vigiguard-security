# VigiPress Security WordPress Plugin

Simple one-click WordPress security. Protect your site in 30 seconds.

## What It Does

- **One-Click Protection** - Fix all security issues with a single button
- **Brute Force Blocking** - Stops automated login attacks
- **File Monitoring** - Detects malware and unauthorized changes
- **Activity Logging** - See who's trying to access your site
- **Zero Performance Impact** - Fast and lightweight

## Installation

1. Download from [WordPress.org]
2. Install via WordPress Admin → Plugins → Add New
3. Activate and click "Fix All Issues"

## Requirements

- WordPress 5.8+
- PHP 7.4+

## Features

### Security Health Score
Get an A-F grade showing your site's security status at a glance.

### Login Protection
- Blocks IPs after failed login attempts
- Configurable attempt limits and lockout duration
- Email notifications

### Security Hardening
- Disables XML-RPC
- Hides WordPress version
- Blocks user enumeration
- Adds security headers

### File Integrity
- Weekly automated scans
- Checks 3,000+ core files
- Email alerts for changes

## Usage

After activation:
```
Dashboard → Click "Fix All Issues" → Done!
```
